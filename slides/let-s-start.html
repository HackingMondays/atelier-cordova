Let's start

Il faut déjà avoir node d'installer. (voir la clé pour ceux qui ne l'ont pas)

Prêt à utiliser (une clé doit être disponible)

ou

  npm install -g cordova

  cordova create MyApp
Attention créé un dossier nommée MyApp 
  cordova platform add browser
Ajoute une plateforme supporté par l'application. Ici le navigateur. 
  cordova run browser
Lance le projet dans le browser (manque l'auto refresh...) 
  cordova plugin add phonegap-plugin-barcodescanner
Ajoute le plugin de scan de QR Codes/Code bar

Vous êtes prêt à faire votre application qui affiche les informations d'un livre
